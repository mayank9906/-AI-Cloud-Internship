# IPL Dataset Analysis

## Objective

Analyze an IPL players dataset stored in a CSV file using Python to identify key player and team statistics.

---

## Dataset

**Dataset Used:** `ipl_players.csv`

The dataset contains information for 100 IPL players, including:

- Player Name
- Team
- Matches
- Runs
- Wickets
- Strike Rate
- Team Wins

---

## Operations Performed

- Read data from a CSV file
- Find the Highest Run Scorer
- Find the Player with the Most Wickets
- Find the Best Strike Rate
- Find the Team with the Most Wins
- Display analysis results

---

## Analysis Performed

### Highest Run Scorer

Identifies the player who has scored the highest number of runs in the dataset.

---

### Most Wickets

Finds the bowler with the highest wicket count.

---

### Best Strike Rate

Determines the player with the highest batting strike rate.

---

### Team with Most Wins

Calculates which IPL team has recorded the highest number of wins.
