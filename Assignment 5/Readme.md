# Student Data Processing

## Objective

Read a CSV dataset, clean the data, and generate summary statistics using Python and the Pandas library.

---

## Dataset

**Dataset Used:** Student Dataset (`students.csv`)

The dataset contains student information such as:

- Student ID
- Name
- Age
- Gender
- Department
- Marks

---

## Operations Performed

- Read CSV file
- Display dataset
- Remove null (missing) values
- Count total records
- Generate summary statistics
- Display cleaned dataset

---

## Technologies Used

- Python 3
- Pandas
- CSV

---

## Files Included

- data_processing.py
- students.csv
- README.md
- Output.png

---

## Requirements

Install Pandas before running the program:

```bash
pip install pandas
```

---

## How to Run

Open Command Prompt in the project folder and execute:

```bash
py -3.14 data_processing.py
```

---

## Expected Output

The program will:

- Read the student dataset.
- Remove rows containing null values.
- Display the total number of records.
- Generate summary statistics.
- Display the cleaned dataset.

---

## Learning Outcomes

- Reading CSV files using Pandas.
- Data cleaning by removing missing values.
- Counting records.
- Generating statistical summaries.
- Working with tabular datasets in Python.
